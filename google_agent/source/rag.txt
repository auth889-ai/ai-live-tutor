. RagRetrievalAgent

This is the source truth retrieval agent.

Its job:

take selected node + chunks
↓
rank exact PDF/resource chunks
↓
produce selectedEvidence / samePageEvidence / nearbyEvidence
↓
produce sourceRefs with page + quote
↓
send rich source truth to Vision / Concept / KG / TeachingStrategy

This agent is not ADK-first because RAG should be deterministic and source-faithful. It should not “imagine” evidence. It is dynamic because it ranks the current node’s real chunks, pages, quotes, and sourceRefs.











. RagRetrievalAgent কী করে?

এই agent-এর কাজ হলো PDF/source থেকে সত্যিকারের evidence বের করা।
